# My Portfolio Website

Welcome to my personal portfolio website showcasing my skills, projects, certifications, education, and contact information.

## About

I am Ouma Joan Akello, a passionate fullstack developer who enjoys crafting beautiful and functional web experiences. This portfolio highlights my journey, expertise, and accomplishments in software and web development.

## Features

- Responsive, modern design with smooth animations
- Dark mode toggle for better user experience
- Sections including About, Education, Skills, Certifications, Projects, and Contact
- Email contact form with mailto integration
- Links to GitHub repositories and live project demos

## Technologies Used

- HTML5, CSS3, JavaScript
- Font Awesome for icons
- Google Fonts (Inter) for typography
- Deployed on Vercel with HTTPS support

## Folder Structure
/static
/images # Profile photos, project, and certification images
/styles # CSS stylesheets
/js
script.js # JavaScript for interactions and animations
index.html # Main HTML page
README.md # Project documentation


## How to Run Locally

1. Clone the repository:
2. Open `index.html` in your favorite browser.
3. No additional dependencies or server required for this static site.

## Deployment

The site is deployed on [Vercel](https://vercel.com) for free HTTPS hosting.
